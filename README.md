# Metaway

## Tecnologias utilizadas 
- Angular@13.2.6
- Bootstrap

## Rodar o projeto
```
git clone https://github.com/Roseffc/Agenda-Pessoal
cd AgendaPessoal
npm install 
ng serve

Abrir navegador: http://localhost:4200
```
